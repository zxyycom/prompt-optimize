# Agent Instructions

## 项目边界

1. 本仓库用于维护 `prompt-optimize` Codex skill。skill 本体放在 `skill/prompt-optimize/`。
2. 根目录和 `docs/` 只承接项目级说明、维护约定、决策记录和仓库元数据。
3. 项目文档只写本仓库内可维护的信息；仓库外路径、临时来源和一次性调查上下文不进入长期文档。

## 维护原则

1. `AGENTS.md` 只保留长期稳定的项目级约定。具体文件职责写在 README 或对应文件中, skill 行为细节写在 skill 本体中。
2. 本项目规模小, 修改前快速扫当前项目文件和相关上下文即可, 不维护固定读取清单。
3. 保持改动范围与用户请求对齐, 不为了整理文档而重写无关内容。
4. 新增长期规则前先判断 owner；只有当前文件适合承接项目级约定时, 才修改 `AGENTS.md`。
5. 正文主要使用中文；除用户要求或目标文件已有语言要求外, 新增内容保持中文。

## 决策记录

1. 影响 skill 行为、引用结构、内容 owner 或长期维护约定的决定, 追加到 `docs/decisions/`。
2. 决策记录应说明问题、取舍、决定和影响, 让后续维护者知道为什么这样做。
3. 低风险文字修正、格式调整和一次性执行细节不需要记录。
4. 新判断替代旧判断时, 追加新记录说明替代关系, 不删除旧记录。
